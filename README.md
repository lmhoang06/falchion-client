## Getting Started

This project uses [Next.js](https://nextjs.org/), [Tailwind CSS](https://tailwindcss.com/), and [Material Tailwind](https://material-tailwind.com/) to build a fast and responsive web application.

### Prerequisites

* Node.js (14 or higher)
* pnpm (latest version)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/your-repo-name.git
```

2. Install dependencies:
```bash
pnpm install
```

3. Start development server:
```bash
pnpm dev
```

4. Open your web browser and navigate to `localhost:7000`
